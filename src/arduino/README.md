HC06
----

Data is send via bluetooth with the HC06 module.

### Connection

Arduino <---> Bluetooth Module

![Cabeling](FXO75JTI3THAGV8.MEDIUM.jpg)

5V to Vcc
GND to GND
D10 to TX
D11 to RX

### Pairing

Name HC-06

PIN 1234
